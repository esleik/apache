# Docker LAMP Environment

## Descripción
Entorno de desarrollo LAMP (Linux, Apache, MySQL, PHP) configurado con Docker y Docker Compose.

Permite levantar un servidor web con Apache y PHP junto a una base de datos MySQL de forma rápida y reproducible.

## Tecnologías
- Docker
- Docker Compose
- Apache
- PHP
- MySQL

## Funcionalidades
- Servidor Apache funcionando en contenedor
- Integración con PHP
- Base de datos MySQL en contenedor
- Configuración mediante docker-compose

## Cómo ejecutar

```bash
docker-compose up -d
